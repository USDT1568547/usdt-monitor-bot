# USDT Monitor Bot 🚀
Tron 블록체인에서 특정 지갑 주소로 들어온 **USDT 입금을 감지**하는 봇입니다.

## 🛠 기능
✅ **USDT 입금 감지 후 Telegram 알림 전송**  
✅ **10초마다 지갑 트랜잭션 조회**  
✅ **GitHub Actions를 이용한 자동 실행 (5분마다)**
